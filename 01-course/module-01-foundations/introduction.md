## ğŸ¯ The 4 Core Elements

| Element | Purpose | Example |
|---------|---------|---------|
| **Instructions** | Define role and task | "You are a senior engineer conducting code review" |
| **Context** | Provide background | "This is a user registration function for a web app" |
| **Input Data** | Give specific content | The actual code to review |
| **Output Format** | Specify structure | "1) Security Issues, 2) Code Quality, 3) Improvements" |

**Key insight:** Every effective prompt uses some or all of these elements!
